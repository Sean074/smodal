## Modal Assurance Criteria.
Unlike the MAC used on the SIMO and MIMO this MAC page is to compare an analytical mode shape and frequency to the experimentally extracted mode shape and frequency adn produce the typical MAC plot.

Note the data used in teh wireframe comparison will be the same data available for MAC.  *bdf of analysis model, and output *.f06.

The user should select which model grids are used to match the measured accelerometer locations.

## Typical MAC plot
 x axis Comparison: experimental mode frequency Hz
 y axis Reference: analytical model mode frequency Hz

Intersecting cell is the MAC for that combination of experimental mode and analytical mode.  The high MAC 1.0 red. Mid MAC .5 green. Low MAC 0.0 dark blue.
